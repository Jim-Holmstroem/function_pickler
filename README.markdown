pickleable
==========
Wrappers to pickle a subset of non-pickable objects in python with the
default pickler.


release name logic
==================
<adjective> <animal>

Where <adjective>[:3] == <animal>[:3]
The <adjactive> rule can be stretched to something that sounds good.

Example
-------

* porous porcupine (v0.1)
* koan koala
